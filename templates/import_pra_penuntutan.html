Import Data Tahapan Pra Penuntutan

Format Khusus: SPDP Data
Panduan Import Data Pra Penuntutan (Format SPDP):
  • File yang didukung: Excel (.xlsx, .xls) dan CSV (.csv)
  • Kolom yang diperlukan: No, Tgl_Nomor, Pasal_yang_Disangkakan
  • Tgl_Nomor berisi tanggal dan nomor SPDP (contoh: "2025-08-28 SPDP/63/VIII/RES.1.24/2025/Reskrim")
  • Pasal_yang_Disangkakan berisi informasi pasal dan UU yang dilanggar
  • Data akan disimpan sebagai tahapan Pra Penuntutan
  • Sistem akan otomatis menganalisis jenis perkara berdasarkan pasal
Format yang didukung: .xlsx, .xls, .csv (Maksimal 10MB)
Kembali ke Input Manual
Template Format File SPDP

Untuk memudahkan import data Pra Penuntutan, gunakan format kolom SPDP berikut:

No Tgl_Nomor Pasal_yang_Disangkakan
1 2025-08-28 SPDP/63/VIII/RES.1.24/2025/Reskrim Pasal 82 Undang-Undang Nomor 17 Tahun 2016 tentang Perlindungan Anak
2 2025-09-02 B/SPDP/65/IX/RES.1.8/Reskrim Pasal 365 KUHP Jo Pasal 55 ayat (1) ke-1 KUHP
3 2025-09-08 SPDP/24/IX/RES.4.2/2025/Resnarkoba Pasal 114 ayat (1) UU RI Nomor 35 Tahun 2009 tentang Narkotika
Fitur Otomatis:
  • Ekstraksi Tanggal: Sistem akan mengambil tanggal dari kolom Tgl_Nomor
  • Analisis Jenis Perkara: Berdasarkan pasal yang disangkakan
  • Kategori Terdeteksi: NARKOBA, PERKARA ANAK, KESUSILAAN, OHARDA, KDRT, JUDI, PERKARA LAINNYA
  • Keterangan Lengkap: Kombinasi nomor SPDP dan pasal
Contoh Deteksi Otomatis:

Pasal Narkoba:

"Pasal 114 ayat (1) UU RI Nomor 35 Tahun 2009" → NARKOBA

Pasal Perlindungan Anak:

"Pasal 82 UU Nomor 17 Tahun 2016" → PERKARA ANAK

Pasal KUHP:

"Pasal 365 KUHP" → OHARDA

Pasal Lainnya:

"Pasal 67 UU Nomor 27 Tahun 2022" → PERKARA LAINNYA
