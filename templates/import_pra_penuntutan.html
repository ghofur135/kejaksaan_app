Import Data Tahapan Pra Penuntutan

Panduan Import Tahapan Pra Penuntutan:
  • File yang didukung: Excel (.xlsx, .xls) dan CSV (.csv)
  • Kolom yang diperlukan: NO, PERIODE, TANGGAL, JENIS PERKARA, KETERANGAN
  • KETERANGAN berisi informasi pasal pelanggaran atau detail lainnya
  • Data akan disimpan sebagai tahapan Pra Penuntutan
  • Setelah upload, Anda dapat memilih jenis perkara yang sesuai untuk setiap data
Format yang didukung: .xlsx, .xls, .csv (Maksimal 10MB)
Kembali ke Input Manual
Template Format File

Untuk memudahkan import tahapan Pra Penuntutan, gunakan format kolom berikut:

NO PERIODE TANGGAL JENIS PERKARA KETERANGAN
1 1 2025-09-30 NARKOBA Pasal 112 UU No. 35 Tahun 2009
2 1 2025-09-30 KESUSILAAN Pasal 289 KUHP
Catatan Jenis Perkara:

Jenis perkara dari file akan dianalisis dan Anda dapat memilih dari kategori yang tersedia: NARKOBA, PERKARA ANAK, KESUSILAAN, JUDI, KDRT, OHARDA, PERKARA LAINNYA
