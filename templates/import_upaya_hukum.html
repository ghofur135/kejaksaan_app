Import Data Tahapan Upaya Hukum

Format Khusus: RP9 Data
Panduan Import Data Upaya Hukum (Format RP9):
  • File yang didukung: Excel (.xlsx, .xls) dan CSV (.csv)
  • Kolom yang diperlukan: No, Terdakwa_Terpidana, No_Tanggal_RP9
  • Kolom opsional: Jenis_Upaya_Hukum, Tanggal_Transaksi, Banding_Akte, Kasasi_Akte, PK_Tanggal
  • No berisi nomor urut data
  • Terdakwa_Terpidana berisi nama terdakwa/terpidana
  • No_Tanggal_RP9 berisi nomor dan tanggal RP9 (contoh: "PDM- 08/PRBAL/Eoh.2/06/2025")
  • Jenis_Upaya_Hukum berisi jenis upaya hukum (Banding, Kasasi, PK)
  • Tanggal_Transaksi berisi tanggal transaksi (contoh: "Terdakwa: 2025-09-08 0")
  • Data akan disimpan sebagai tahapan Upaya Hukum
  • Sistem akan otomatis menganalisis jenis perkara berdasarkan informasi terdakwa
Format yang didukung: .xlsx, .xls, .csv (Maksimal 10MB)

Template Format File

Berikut contoh format file CSV yang diharapkan:

No Terdakwa_Terpidana No_Tanggal_RP9 Jenis_Upaya_Hukum Tanggal_Transaksi Banding_Akte Kasasi_Akte PK_Tanggal
1 FADELA TRI ANGGRAENI Alias DELA Binti PAIJO PDM- 08/PRBAL/Eoh.2/06/2025 Banding Terdakwa: 2025-09-08 0 Terdakwa: 2025-09-08 0
2 RENO ALI MUKHTAMAR Alias RENO Bin AHMAD SUBARNO PDM- 11/PRBAL/Eoh.2/07/2025 Banding Terdakwa: 2025-09-22 0 Terdakwa: 2025-09-22 0
3 ADITYA SETIAWAN bin WAHIDIN PDM- 19/PRBAL/Enz.2/06/2025 Banding Jaksa: 2025-09-01 0 Jaksa: 2025-09-01 0
Catatan Penting:
  • Kolom wajib: No, Terdakwa_Terpidana, No_Tanggal_RP9
  • Kolom opsional: Jenis_Upaya_Hukum, Tanggal_Transaksi, Banding_Akte, Kasasi_Akte, PK_Tanggal
  • Pastikan nama kolom tepat sama dengan template di atas
  • Jangan ada baris kosong di tengah data
  • Nama terdakwa harus diisi dengan lengkap
  • Nomor RP9 harus mengikuti format standar dengan tanggal di akhir
  • File CSV harus menggunakan encoding UTF-8
  • Sistem akan menggunakan Tanggal_Transaksi jika tersedia, atau mengekstrak dari No_Tanggal_RP9
