Import Data Tahapan Upaya Hukum

Format Khusus: RP9 Data
Panduan Import Data Upaya Hukum (Format RP9):
  • File yang didukung: Excel (.xlsx, .xls) dan CSV (.csv)
  • Kolom yang diperlukan: No, Terdakwa_Terpidana, No_Tanggal_RP9
  • No berisi nomor urut data
  • Terdakwa_Terpidana berisi nama terdakwa/terpidana
  • No_Tanggal_RP9 berisi nomor dan tanggal RP9 (contoh: "PDM- 08/PRBAL/Eoh.2/06/2025")
  • Data akan disimpan sebagai tahapan Upaya Hukum
  • Sistem akan otomatis menganalisis jenis perkara berdasarkan informasi terdakwa
Format yang didukung: .xlsx, .xls, .csv (Maksimal 10MB)

Template Format File

Untuk memudahkan import tahapan Upaya Hukum, gunakan format kolom berikut:

NO PERIODE TANGGAL JENIS PERKARA KETERANGAN
1 1 2025-09-30 NARKOBA Pasal 112 UU No. 35 Tahun 2009
2 1 2025-09-30 KESUSILAAN Pasal 289 KUHP
Catatan Jenis Perkara:

Jenis perkara dari file akan dianalisis dan Anda dapat memilih dari kategori yang tersedia: NARKOBA, PERKARA ANAK, KESUSILAAN, JUDI, KDRT, OHARDA, PERKARA LAINNYA
