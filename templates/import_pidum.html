Import Data PIDUM dari File Excel/CSV

Panduan Import:
  • File yang didukung: Excel (.xlsx, .xls) dan CSV (.csv)
  • Pastikan file memiliki kolom: NO, PERIODE, TANGGAL, JENIS PERKARA, PRA PENUTUTAN, PENUNTUTAN, UPAYA HUKUM
  • Setelah upload, Anda dapat memilih jenis perkara yang sesuai untuk setiap data
  • Data akan ditampilkan untuk preview sebelum disimpan ke database
Format yang didukung: .xlsx, .xls, .csv (Maksimal 10MB)
Kembali ke Input Manual
Template Format File

Untuk memudahkan import, gunakan format kolom berikut:

NO PERIODE TANGGAL JENIS PERKARA PRA PENUTUTAN PENUNTUTAN UPAYA HUKUM
1 1 2025-09-30 NARKOBA 5 3 2
2 1 2025-09-30 KESUSILAAN 2 1 1
Catatan Jenis Perkara:

Jenis perkara dari file akan dianalisis dan Anda dapat memilih dari kategori yang tersedia: NARKOBA, PERKARA ANAK, KESUSILAAN, JUDI, KDRT, OHARDA, PERKARA LAINNYA
